#### Ubuntu上安装 树莓派4 64位 交叉编译环境

```bash
# ELF 64-bit LSB pie executable, ARM aarch64
$ sudo apt update -y
$ sudo apt install -y gcc-aarch64-linux-gnu g++-aarch64-linux-gnu build-essential
$ sudo apt install gdb-multiarch    # 远程调试异构架构
```
